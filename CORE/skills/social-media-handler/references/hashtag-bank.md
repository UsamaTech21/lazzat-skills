## Hashtag Rules

1. **3–5 in-caption.** IG punishes 20+.
2. **Rotate.** If you used the same set last week, swap at least 2.
3. **Mix sizes.** One broad (#Brampton), one mid (#BramptonFood), one niche (#LavaStoneGrill).
4. **Relevance beats volume.** Unrelated trending tag = flag.
5. **No hashtag stuffing in comments either.** Algorithm detects it.
6. **Never hashtag a competitor.**
7. **Seasonal tags expire** — remove after the event window.
