# Caption Templates

> **Rule:** Every caption ends with a question, carries one pillar, names one item or strength, and passes compliance.

---

## Template 1 — Menu Spotlight

```
[Strong noun-first hook — 5–8 words]

[Item] on lava stone, plated over [base] with [sauce or salad]. [One-line brand truth — global framing / dietary inclusivity / 200g honesty.]

[Question — what pairs, what they'd add, which sauce?]

📍 [Location]
```

**Example — Chicken Cube:**
```
One cube. Four ways to finish it.

Lava stone Chicken Cube, 200g honestly weighed, over Saffron Rice with Lazzat Green.
Every table welcome. No pork. No beef. No alcohol.

Which sauce — Green, White, or Red?

📍 McVean Dr, Brampton
```

## Hook Starters (if you're stuck)

- "One cube. [Three ways…]"
- "We replaced [thing] with [better thing]."
- "[Number] sauces. One for every [thing]."
- "POV: You just walked into…"
- "Today's table: [family name]"
- "[Time], [location] — [action]"
- "No [banned thing]. No [banned thing]. Just [our thing]."
- "The reason we [don't do X]."
- "Watch this [specific action] in slow-mo."
- "Built for every table that [specific moment]."

---

## Anti-Hook List (never start a caption with)

- "We are delighted to…"
- "Hey everyone…"
- "Today we're going to talk about…"
- "In this post…"
- "Lazzat is a…"
- "Welcome to…"
- "Check out our new…"
- Any emoji as the first character
- Any hashtag as the first character
