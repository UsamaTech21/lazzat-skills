## Every Post — Non-Negotiable Rules

1. **Caption ends with a question.** Always. Drives comments, drives algorithm.
2. **Location tag on every post.** "Brampton" or "McVean Dr, Brampton" or "Clarence St, Brampton."
3. **Exactly ONE pillar per post.** Multi-pillar = rewrite.
4. **One strength + one menu item per post.** Don't stack three.
5. **First 30 days post-launch:** reply to every comment within 2 hours.
6. **Compliance block attached** at the end.

## DM Playbook

Draft every DM reply as:
1. **Acknowledge** — person by name (if in bio) + specific reference ("Thanks for the order Friday night!")
2. **Answer** the question directly in 1–2 sentences
3. **Invite** a next step (revisit, try X, share with friend, leave review)
4. **Sign off** warm but short

**Auto-escalate to Usama if DM contains:**
- Complaint (food safety / allergen / religion / discrimination)
- Press / media request
- Catering > $500 inquiry
- Business partnership
- Influencer pitch

---

## Review Reply Playbook

- **5★ review:** Thank by name + feature one menu item they mentioned + invite return. ≤ 2 sentences.
- **4★ review:** Thank + ask what would have made it 5 (only if they left room to improve). ≤ 3 sentences.
- **3★ review:** Sincere apology + specific fix + offer a direct line (Soha's DM or email). Usama must pre-approve.
- **1–2★ review:** **Hold + escalate to Usama within 15 minutes**. Never reply in anger. Never templated.
- **Never:** delete a review, argue publicly, offer money without Usama, or use the word "unfortunately."

---

## Comment Reply Playbook

- Reply to every comment in the first 48 hours (launch window: 2 hours).
- Heart + reply with ≥ 3 words. "❤️" alone is lazy.
- Tag back only if it adds value. Don't overtag.
- If commenter reveals customer name/moment → ask in DM for consent to feature.
