- All reporting: **Monday 9am dashboard to Lazzat-Brand**

- Seasonal launches: **5–7 days before event**
- Pause any campaign with CPA > 2× target for 3 consecutive days — reassess

## Do-Not-Run Windows

| Window | Reason |
|---|---|
| Yom HaShoah / Holocaust Remembrance Day | No promotional ads |
| Major national tragedy / Canadian mourning day | Pause all commercial ads for 24–48h |
| Ramadan daylight hours | No food-hero creative 11am–6pm in Muslim-language targeting; move to evening |
| Good Friday / Easter Sunday | No promotional creative; brand-silent |
| Day of a local Brampton tragedy | Pause all ads immediately |
