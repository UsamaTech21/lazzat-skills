# Post Templates — Canva Layout Library

> Every template here is design-locked. To change a layout pattern, route through Usama.

## Template 1 — Hero Dish (Charcoal)

**Use:** menu launches, signature dish spotlights
**Size:** 1080×1080 (Feed) and 1080×1350 (Feed 4:5)

```
LAYOUT:
- Background: charcoal #1A1A1A
- Hero image: 70% frame, centered, food 45° angle
- Top-left: white wordmark logo (120 px)
- Bottom-left: headline (Arial Black, white, 72pt) — ≤ 5 words
- Bottom-right: CTA chip — red #C0392B, white text, "Order Now"
- Subtle gold rule line (2 px) separating headline from image

CONTENT RULES:
- Headline: dish name + 1 differentiator ("Lamb Cube. 200 grams.")
- No body text
- Menu item must appear on menu-v14.md
```

## Template 3 — Carousel (3-slide + CTA)

**Use:** dish explainers, how-it-works, 13 sauces guide
**Size:** 1080×1350 (Feed 4:5)

```
SLIDE 1 — HOOK
- Charcoal background
- Big headline (Arial Black white, 120pt): the question or promise
- Small "Swipe →" text bottom-right (gold)

SLIDE 2 — ANSWER / STEP 1
- Cream background
- Supporting visual or icon (top 60%)
- Short body (Inter charcoal, 32pt, ≤ 30 words)

SLIDE 3 — ANSWER / STEP 2
- Charcoal background
- Hero dish photo (top 60%)
- Short body (Inter white, 32pt)

SLIDE 4 — CTA
- Cream background
- Tagline in gold handwritten script
- CTA chip red #C0392B: "See Menu" / "Book Catering"
- Wordmark logo bottom-center

CONTENT RULES:
- Max 4 slides per carousel (prevents drop-off)
- Each slide has only ONE focal element
- No slide has more than 30 words
```

## Template 4 — Story (Vertical 9:16)

**Use:** daily BTS, updates, reposts, polls
**Size:** 1080×1920

```
LAYOUT:
- Full-frame image (BTS shot / dish / team)
- Top 250px: reserved for IG UI (username, timer)
- Middle 60%: IMAGE ZONE — nothing critical outside here
- Bottom 250px: reserved — logo bottom-right 80×80, tagline script small above it
- Text overlay: center of middle 60%, max 10 words

CONTENT RULES:
- One story per post (never layered info dumps)
- Readable in 3 seconds
- Polls / stickers: anchor to middle 60%, never in UI-reserved zones
```

## Template 7 — Menu Board (Digital / Print)

**Use:** in-store menu display
**Size:** 1920×1080 (digital screen); A2 (print 420×594mm)

```
LAYOUT:
- Charcoal background
- Top strip: wordmark logo left + "Lazzat Menu — v14" right (gold small text)
- Body: 2-column grid
  - Left column: Platters + Wraps + Bowls
  - Right column: Kids + Combos + Drinks + Sides + Desserts

CONTENT RULES:

- Blocked items (per menu-v14) must NOT appear
```
