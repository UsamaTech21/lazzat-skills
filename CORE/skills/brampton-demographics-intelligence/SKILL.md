## The Golden Rule (non-negotiable)

Use community data to decide **WHEN** and **WHAT** to feature. **NEVER** to say Lazzat is FOR a specific community.

> "Lazzat is for everyone. The calendar helps us decide which story to tell that week."

Any piece that implies Lazzat is "a Pakistani restaurant" / "for the Sikh community" / "halal place" / "Hindu-friendly" is an automatic REWRITE.

## Location Audiences

### McVean Dr (Castlemore)

- **Neighbourhood:** Castlemore — high-income, family-oriented, deeply multicultural
- **Median household income:** significantly above Brampton average
- **Primary audiences:**
  - Young families (kids 0–12) — weekday dinner + weekend family platter
  - Upwardly mobile professionals (28–45) — weekday lunch + delivery dinner
  - Multicultural extended-family gatherings — weekend platter + catering
- **Driving radius:** 3km primary, 5km secondary (see `references/area-map.md`)

### Clarence St (Downtown)

- **Neighbourhood:** Downtown Brampton — mixed commercial + residential + transit hub
- **Primary audiences:**
  - Downtown workers (28–55) — weekday lunch high-volume
  - Transit commuters (18–40) — dinner grab-and-go, delivery pickup
  - Students + young adults (18–30) — evening + late-night
  - Local business catering (offices, clinics)
- **Driving radius:** 3km primary, 5km secondary

## Handling Festivals — The Sensitivity Protocol

For every festival:

1. **Is the festival religious?** Yes → extra care with imagery, no religious symbols in marketing.
2. **Is the festival culturally sensitive in any direction?** (e.g., Navratri = many Gujarati Hindus fast meat for 9 days) → adjust featured menu for that window.
3. **Is the framing global?** A Diwali post should say "Light up your table" not "Happy Diwali to our Hindu customers."
4. **Menu pairing:** Pull from `menu-community-match.md` — right item, right week.
5. **Visual tone:** Festive but never costume-y. No marigolds, diyas-on-banana-leaves, mirror-work props.
6. **Compliance:** Every festival post routes through brand-compliance-checker with special attention to §1 (regional) and §6 (religious).

## The Decision Tree (when a teammate asks "should we run X for Y community?")

```
Is the request for Lazzat to target a single community exclusively?
  → NO. Reframe as global + festival-aware.

Is there a Brampton cultural festival in the next 14 days?
  → YES. Check cultural-calendar.md → propose pillar + menu item → draft brief.

Is the ask "who do we target in Castlemore vs Downtown"?
  → Use area-map.md — community-agnostic neighbourhood logic.

Is the ask "which menu item for X community"?
  → menu-community-match.md — always pairs with a global-framing caption.

Is the ask about a sensitive window (fasting, religious observance)?
  → Apply sensitivity protocol. Often the right answer is "feature vegetarian/vegan" or "quiet that week."
```
