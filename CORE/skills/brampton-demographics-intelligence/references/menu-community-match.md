# Menu ↔ Community Match

> **Rule:** Pair the right item with the right community moment. Never say "this item is FOR X community." Always frame: "this item pairs with [moment]."

---

## Pairing Logic

1. Identify the week's moment (festival / event / season).
2. Identify the community most connected to it.
3. Pull the item from `menu-v14.md` that naturally pairs.
4. Draft with global framing ("Every table, today's table is [moment]").
5. Run compliance → post.

## Rules When Pairing

1. **One item + one community moment per post.** Don't stack three.
2. **Global framing always.** Never say "built for X community."
3. **Respect sensitivities.** During Navratri → no meat hero. During Ramadan day → no food imagery in daylight.
4. **Rotate.** Every community should be naturally featured at least once a month across the six pillars.
5. **Cross-pollinate.** A sauce from one heritage on a protein from another — that IS Lazzat. Celebrate the union.
