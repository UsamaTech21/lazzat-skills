# ALWAYS-DO List — The Positive Rules

> **STATUS:** Locked
> **Owner:** Lazzat-Brand
> **Rule:** Every piece must be scanned for what's MISSING, not only what's wrong.

A piece passes DO-NOT-DO but still fails ALWAYS-DO if it forgets what Lazzat is.

---

## 1. Global Brand Framing

**Required when the piece talks about who Lazzat serves.**

| Good examples |
|---|
| "Every culture welcome." |
| "Built for every table." |
| "One grill. Every story." |
| "Uniting through flavours." |
| "No pork. No beef. No alcohol. Every plate fits every table." |

A piece that only features one community without the global frame → REWRITE with global frame added at the start or end.

---

## 2. Lava Stone Angle

**Required any time the grill, method, or cooking is mentioned.**

| Good examples |
|---|
| "Lava stone grilled — real flame, no charcoal." |
| "Volcanic stone, not coal." |
| "From our lava stone to your table." |
| "Eco-friendlier method: no carbon ash, low-oil." |

A reel showing the grill with no lava stone mention → REWRITE the caption to include it.

---

## 3. Dietary Inclusivity

**Required in any "who can eat here" piece.**

| Good examples |
|---|
| "Halal-sourced. Vegetarian. Vegan. All on one menu." |
| "Paneer Cube, Tofu Cube, Falafel Wrap — plant-forward options always." |
| "No pork. No beef. No alcohol. Every table fits." |

---

## 4. Content Pillar Assignment

**Every piece must map to ONE of the 6 pillars.**

1. Lava Stone Story
2. Menu Spotlight
3. Global Kitchen
4. Team & Craft
5. Community Brampton
6. Wellness & Lifestyle

If no pillar fits → the piece is off-strategy → send back to strategist.

---

## 5. Tagline Usage

**Long-form and in-store pieces** should use the primary tagline: *Uniting Through Flavours*.
**Short-form, ads, and delivery-platform copy** should use the secondary: *Taste the World. Crafted Fresh. Served Fast.*

Every brand film ending — primary tagline. Every paid ad — secondary.

---

## 6. Menu Accuracy

Every named dish must be on `menu-v14.md`. Every sauce must be one of the 13. Every format must be Cube / Kabab / Wrap / Platter.

---

## 7. Location + Community Anchor

**Required on Community Brampton pillar pieces** and on any ad.

| Required mention |
|---|
| "McVean Dr, Castlemore, Brampton" |
| "Clarence St, Downtown Brampton" |
| Brampton (the city) named once |

Missing location = missing local SEO + local community signal.

---

## 8. Approver Flagged

Every delivered piece ends with the correct approver named:

- Routine post after PASS → Lazzat-Brand + Lazzat-Social
- New campaign → Lazzat-Founder
- Paid ad > $100 → Lazzat-Founder
- Blog → Lazzat-Brand
- Wall panel → Lazzat-Brand + Lazzat-Founder
- Menu change → Lazzat-Founder

Missing approver = incomplete output.

---

## 9. Call-to-Action (CTA)

**Required on every published piece.**

| Piece type | Appropriate CTA |
|---|---|
| Social caption | Question to the audience (drives comments) |
| Reel / TikTok | Visual CTA + caption hook |
| Blog | "See the menu" / "Visit McVean" |
| Ad | "Order now" / "See menu" |
| Email | One ask — no more |
| DM reply | One next step |
| Wall panel | Physical-world action (scan QR, ask the chef, etc.) |

A piece with no CTA → REWRITE with CTA.

---

## 10. Compliance Block Attached

Every output ends with the structured Compliance Block. No exceptions.
