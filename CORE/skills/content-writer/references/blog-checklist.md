# Blog Publishing Checklist

> **Rule:** Every blog runs through this end-to-end before it lands in Lazzat-Brand's inbox for approval.

---

## Stage 1 — Topic + Keyword Lock

- [ ] Topic picked from `topic-bank.md` OR approved by Lazzat-Brand if new
- [ ] Primary keyword picked from `CORE/skills/seo-expert/references/keyword-targets.md`
- [ ] Secondary keywords (2–3) listed
- [ ] Search intent identified: informational / navigational / commercial / local
- [ ] Audience identified: first-time visitor / repeat customer / catering lead / local search

---

## Stage 2 — Outline

- [ ] H1 includes primary keyword (and reads naturally)
- [ ] 3–6 H2s drafted before prose
- [ ] Each H2 has a rough word-count budget
- [ ] Brand-story element identified (which of the 5 brand promises to thread?)
- [ ] Pillar assigned (1 of 6)

---

## Stage 3 — Draft

### Opening hook (first 80 words)
- [ ] Noun-first hook (not "We" / "Our" / "Let's")
- [ ] Primary keyword appears in first 100 words
- [ ] Tension or question drives into H2s

### Body
- [ ] Each H2 section 100–250 words
- [ ] At least 1 real proper noun per section (Brampton / McVean / chef name / ingredient)
- [ ] No banned words (see `tone-guide.md`)
- [ ] No unapproved claims
- [ ] Every menu mention matches `menu-v14.md` exactly
- [ ] Every claim appears on `approved-claims.md` GREEN list

### Closer
- [ ] One brand truth (which of the 5 promises)
- [ ] Primary tagline appears ONCE (long-form pieces only)
- [ ] One CTA — "Visit McVean Dr" / "See the menu" / "Book catering"
- [ ] Ends with a question or a clear next step

---

## Stage 4 — SEO Meta

- [ ] Title tag ≤ 60 chars, includes primary keyword + brand
- [ ] Meta description 150–160 chars, includes keyword + CTA
- [ ] URL slug short, lowercase, hyphenated, keyword-focused
- [ ] Featured image brief (for graphic-designer)
  - Aspect 1200×628 preferred for blog share
  - Style: warm cream flat-lay OR charcoal hero (brand palette)
  - Alt text drafted (includes keyword + dish name)
- [ ] Internal links (min 2):
  - Link to `/menu`
  - Link to `/locations` or `/catering`
- [ ] External links (min 1, trusted source — e.g., Brampton.ca, Statistics Canada) if factual claim used

---

## Stage 5 — Tone + Brand Compliance

- [ ] Tone-guide word list applied (no "passion", "authentic", "fusion", "journey", "delightful", etc.)
- [ ] No regional framing (Pakistani / Desi / Indian / South Asian — all banned)
- [ ] No charcoal / BBQ / tandoor language
- [ ] No speed / freshness / health superlatives
- [ ] No religious phrases (JazakAllah / Inshallah)
- [ ] "Halal-sourced" never "halal-certified"
- [ ] Team quotes verified against `team-bios.md` (no invented quotes)
- [ ] All 6 pillar cross-check (which does this primarily serve?)

---

## Stage 6 — Structural QA

- [ ] Word count 700–1,200 (standard) or 1,200–2,500 (pillar)
- [ ] Paragraphs vary in length (no 5 identical-length paragraphs in a row)
- [ ] No paragraph longer than 4 sentences
- [ ] No em-dash walls (max 2 em-dashes per 200 words)
- [ ] No passive voice clusters (scan for "was / were / been" — rewrite to active)
- [ ] No "very" / "really" filler adverbs

---

## Stage 7 — Readability + Accessibility

- [ ] Grade 8–10 reading level (flesch-kincaid ~60–75)
- [ ] Lists used when information is list-shaped
- [ ] Tables used when comparison-shaped
- [ ] Image alt text descriptive (not decorative)
- [ ] Heading hierarchy clean (H1 → H2 → H3, no skips)

---

## Stage 8 — Compliance Block

- [ ] Invoke `brand-compliance-checker`
- [ ] Block shows ✅ PASS
- [ ] Pillar named
- [ ] Menu items listed
- [ ] Approved claims listed
- [ ] Tagline usage declared
- [ ] Approver named (Lazzat-Brand for routine; Lazzat-Founder for anything touching strategy)

---

## Stage 9 — Hand-off Package

Deliver to Lazzat-Brand as one document with:

1. Blog (with frontmatter)
2. Featured image brief (for graphic-designer)
3. 3 social caption teasers (for social-media-handler to repurpose)
4. 1 newsletter teaser (for newsletter)
5. Compliance Block

---

## Stage 10 — Post-Publish

- [ ] URL submitted to Google Search Console
- [ ] URL shared with Lazzat-SEO (SEO tracking)
- [ ] 3 social teasers queued in Later / Meta Business Suite
- [ ] Newsletter teaser added to next newsletter draft
- [ ] Blog added to internal VERSION tracker

---

## Common Rejection Reasons (and how to fix)

| Reason | Fix |
|---|---|
| "Feels generic" | Add 2+ Brampton-specific proper nouns |
| "Sounds corporate" | Rewrite the opener to noun-first, cut all hedging |
| "Keyword stuffed" | Reduce keyword density below 2%; use natural variations |
| "Off-pillar" | Reassign pillar or redirect the blog to fit |
| "Missing compliance block" | Run the checker |
| "Too long" | Cut whichever H2 is closest to redundant |
| "No CTA" | Add a clear next step in the closer |
